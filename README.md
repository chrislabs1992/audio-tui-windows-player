## What is AUDIO.TUI?

AUDIO.TUI is built for people who want to handle audio from the terminal without giving up convenience. It combines local music playback and internet radio access with browsing and visual feedback, all within one compact command-line interface.

The experience is aimed at keyboard-focused users who prefer a lightweight desktop-free workflow. By keeping file browsing and system audio support close to playback controls, it stays practical for both personal libraries and streaming sources.
